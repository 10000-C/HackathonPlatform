## 项目概述 (Hackthon Platform)

这是一个基于以太坊的去中心化黑客松管理平台智能合约系统。该平台允许组织者创建和管理黑客松活动，参与者提交项目演示，评委对项目进行评分，以及管理员进行奖项分配。
